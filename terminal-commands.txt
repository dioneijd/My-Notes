pwd >>> mostra caminho atual

cd >>> change directory - altera a pasta atual

whoami >>> mostra o usuario logado

ls >>> lista todos os arquivos e pasta do diretorio atual
ls -a >>> lista todos os arquivos e pasta do diretorio atual mesmo que esteja oculto

echo "Bem Vindo" >>> mostra mensagem na tela

echo "Bem Vindo" > arquivo.txt >>> gera um arquivo texto com a mensagem Bem vindo
